Cars API URL endpoints

CarList

"GET POST /api/v1.2/car/ HTTP/1.1"

Show all car info

CarAPIView

"GET /api/v1.1/car/ HTTP/1.1"

POST /api/v1.1/car/ JSON example
{ "category_id": 1, "manufacturer_id": 1, "model_name": "Fiesta", "description": "The Ford Fiesta is a supermini car." }

Show all car info

CarViewSet

"GET /api/v1.5/car/ HTTP/1.1"

"PUT /api/v1.5/car/<int>" JSON example

Uses a ViewSet
